# @tensordoc/prism — CodeSandbox source

This directory contains the three files that live in the CodeSandbox
demo: `index.html`, `main.js`, `package.json`. The sandbox just runs
them through Vite.

To rebuild the sandbox from these files: paste each one into a fresh
CodeSandbox Vite vanilla template (codesandbox.io/p/sandbox/vite-vanilla).
